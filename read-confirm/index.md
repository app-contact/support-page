# Read Confirm - Documentation

## Overview

Read Confirm helps Confluence admins track document acknowledgments and manage confirmation workflows. Verify that employees have read and acknowledged important policies, guidelines, or other critical documents with auditable confirmation records.

## Getting Started

1. Navigate to **Apps** â†’ **Read Confirm** in Confluence
2. Click **Create Request** to set up a new confirmation request
3. Select target users or choose "All Viewers" mode
4. Optionally set a deadline and message
5. The confirmation macro is automatically embedded on the target page

## Features

### Confirmation Request Management

Create confirmation requests with flexible targeting options:
- **Individual Users**: Select specific people who must confirm reading
- **All Viewers**: Automatically track anyone who views the page

### Embedded Confirmation Macro

Users can confirm directly from the page where the document is:
- One-click confirmation button
- Progress tracking display (e.g., "5/8 confirmed")
- Status indicators for confirmed/pending
- Deadline and message display

### Administrator Dashboard

Central management hub for all confirmation requests:
- List view with pagination
- Filter by status (active/completed/archived)
- Quick access to detailed pages
- Refresh page titles when pages are renamed

### Detailed Tracking

- View confirmed and pending user lists
- See confirmation timestamps
- Progress visualization with progress bars
- CSV export for audit purposes (STANDARD+ plans)
- Send reminders to pending users (STANDARD+ plans)

## Requirements

- Confluence Cloud
- Appropriate Confluence permissions

## FAQ

**Q: How does the "All Viewers" mode work?**
A: When enabled, the app automatically registers anyone who views the page as a pending confirmation. Users can then confirm by clicking the confirmation button on the page.

**Q: Does this app store any data?**
A: Yes. The app stores confirmation request data and confirmation records using Atlassian Forge Storage. This data is stored securely within Atlassian's infrastructure and is not transmitted externally.

**Q: Can I export confirmation records?**
A: Yes, users on STANDARD and ENTERPRISE plans can export confirmation records as CSV files for audit purposes.

**Q: How do I send reminders to users who haven't confirmed?**
A: The reminder feature is available on STANDARD and ENTERPRISE plans. Navigate to the request detail page and click "Send Reminder" to notify pending users.

**Q: What permissions does the app require?**
A: The app requires read access to Confluence pages, users, and groups, plus write access to embed the confirmation macro on pages.
